Code for BTV

A Code for America Brigade
based in Burlington, Vermont

Who we are

Code for BTV is a group of volunteers who unite with community-focused organizations in Vermont to help them serve the public good with the efficient, effective, and equitable use of modern technology. We do this because we feel that improper use of technology is a social justice issue.
